# Desk-savvy
DeskSavvy is a comprehensive web-based office management system designed to streamline employee and administrative tasks efficiently. The system provides a centralized platform for managing employee records, leave requests, file sharing, and admin controls, ensuring smooth office operations.
Built using Java Servlets, JSP, and MySQL, DeskSavvy offers a user-friendly interface with secure authentication, role-based access, and modern UI enhancements using Bootstrap and JavaScript.
